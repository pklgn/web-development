Функция calc работает рекурсивно, то есть она проходит по выражению слева направо и ожидает получить 
выражение, которым является последовательность из оператора и двух операндов
Перед этим выражение из строки преобразуется в массив символов (числа приводятся к типу Number)
Рекурсивный подход помогает осуществлять проверку на правильность введенного выражения (количество скобок)
Неправильные символы проверяются на этапе преобразования строки в массив
